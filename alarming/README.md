# Alarming

A premium alarm clock app that combines beautiful design, comprehensive time management features, and daily gamified wake-up challenges.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK (>=3.2.0)
- Dart SDK

### Setup

1.  **Install Dependencies:**
    ```bash
    flutter pub get
    ```

2.  **Run the App:**
    ```bash
    flutter run
    ```

## Project Structure

-   `lib/core`: Core utilities, theme, router.
-   `lib/features`: Feature-based modules (home, alarm, etc.).
-   `lib/main.dart`: Application entry point.

## Features (MVP)

-   Dark-first UI design.
-   Home screen with clock and next alarm preview.
-   Basic navigation structure.
